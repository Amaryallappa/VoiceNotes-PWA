# VoiceNotes PWA 🎙️

A lightweight, offline-first Progressive Web App for recording voice notes, text, images, and files for project management. 

I built this because I needed a simple, fast way to gather information while working on projects, but existing solutions were heavily paywalled. Once a project is mapped out, you can export all your notes as a single offline HTML file to feed into AI agents for project context.

## Features
* **100% Local & Private:** All data is saved on your device using IndexedDB. No servers, no accounts.
* **Offline Capable:** Install it as a PWA and use it anywhere.
* **Rich Media:** Supports speech-to-text, typing, camera/gallery images, videos, and file attachments (PDF/TXT/HTML).
* **Exportable:** One-click export of a whole project into a clean, standalone HTML file.
